You do 8 on... events. We have done onClick and onMouseOver. The `onchange` event occurs when **the value of an HTML element is changed**. This event is similar to the [`oninput`](https://www.w3schools.com/jsref/event_oninput.asp) event. The difference is that the `oninput` event occurs immediately after the value of an element has changed, while `onchange` occurs when the element loses focus, after the content has been changed.
