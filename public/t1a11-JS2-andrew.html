Event Description
onChange: occurs when the value of an HTML element is changed.
onMouseOut: occurs when the mouse pointer moves out of an element.
onKeyDown occurs when the user presses a key on the keyboard.
onLoad: occurs when an object has been loaded.
onScroll: occurs when an element's scrollbar is being scrolled.
onSubmit: occurs when a form is submitted.
onSelect: occurs after some text has been selected in an element.
onBlur: occurs when an HTML element loses focus.
